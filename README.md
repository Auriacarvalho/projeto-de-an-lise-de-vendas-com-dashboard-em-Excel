# 📊 Análise de Vendas com Excel

Projeto de análise de dados desenvolvido em Excel utilizando tabela dinâmica e dashboard interativo.

## Objetivo
Analisar dados de vendas e criar visualizações que auxiliem na tomada de decisão.

## Ferramentas utilizadas
- Microsoft Excel
- Tabela Dinâmica
- Segmentação de Dados
- Dashboard Interativo

## Indicadores analisados
- Total de vendas
- Produtos mais vendidos
- Distribuição de vendas
- Análise por categoria
